odtoolkit.stats package
=======================

This package contains functions for analyzing ``odtoolkit.data.dataset.Dataset`` and assessing its quality.
:func:`odtoolkit.stats.analysis.analysis` runs all sorts of analysis automatically.

.. py:function:: odtoolkit.stats.dropout_rate.dropout_rate(dataset, dataset_level=False)
    :noindex:

    :func:`~odtoolkit.stats.dropout_rate.dropout_rate` function computes the percentage of data points missing in
    the ``odtoolkit.data.dataset.Dataset`` object.

.. py:function:: odtoolkit.stats.frequency.frequency(dataset, dataset_level=True)
    :noindex:

    :func:`~odtoolkit.stats.frequency.frequency` function computes the average sampling frequency of all
    features in the ``odtoolkit.data.dataset.Dataset`` object.

.. py:function:: odtoolkit.stats.gap_detect.gap_detect(dataset, threshold, sensor_level=False)
    :noindex:

    :func:`~odtoolkit.stats.gap_detect.gap_detect` function returns all **gaps** in the ``odtoolkit.data.dataset.Dataset`` object.

    We say that there is a **Gap** when the difference between timestamps of two successive values
    exceeds some **threshold**.

.. py:function:: odtoolkit.stats.occupancy_evaluation.occupancy_distribution_evaluation(dataset, dataset_level=True)
    :noindex:

    :func:`~odtoolkit.stats.occupancy_evaluation.occupancy_distribution_evaluation` function computes the distribution of
    different occupancy states in the ``odtoolkit.data.dataset.Dataset`` object.

.. py:function:: odtoolkit.stats.uptime.uptime(dataset, threshold, gaps=None)
    :noindex:

    :func:`~odtoolkit.stats.uptime.uptime` function computes the length of time a sensor reported values. It differs from
    the :func:`~odtoolkit.stats.dropout_rate.dropout_rate` in that it captures the length of the time interval in which
    data points were available in the ``odtoolkit.data.dataset.Dataset`` object.

A sample code snippet for using the package is here::

    >>> import odtoolkit
    >>> dataset = odtoolkit.data.load_sample("umons-all")
    >>> from pprint import pprint
    >>> # ===== Test dropout rate =====
    >>> pprint(odtoolkit.stats.dropout_rate(dataset, dataset_level=False))
    {'datatest': 0.0, 'datatest2': 0.0, 'datatraining': 0.0}
    >>> pprint(odtoolkit.stats.dropout_rate(dataset, dataset_level=True))
    0.0
    >>> # ===== Test frequency =====
    >>> pprint(odtoolkit.stats.frequency(dataset, dataset_level=False))
    {'datatest': 60.0, 'datatest2': 60.0, 'datatraining': 60.0}
    >>> pprint(odtoolkit.stats.frequency(dataset, dataset_level=True))
    60.0
    >>> # ===== Find all gaps =====
    >>> pprint(odtoolkit.stats.gap_detect(dataset, 65, sensor_level=False))
    {'datatest': [], 'datatest2': [], 'datatraining': []}
    >>> pprint(odtoolkit.stats.gap_detect(dataset, 65, sensor_level=True))
    {'datatest': {'CO2': [],
