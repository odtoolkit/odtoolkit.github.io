odtk.preprocessing package
==========================

This package contains functions that are useful for modifying values in ``odtk.data.dataset.Dataset``.
:func:`odtk.preprocessing.auto_clean.auto_clean` automatically runs all preprocessing functions.

odtk.preprocessing.change module
--------------------------------

.. py:function:: odtk.preprocessing.change.change_to_binary(dataset)
    :noindex:
.. py:function:: odtk.preprocessing.change.change_to_label(dataset)
    :noindex:
.. py:function:: odtk.preprocessing.change.change_to_one_hot(dataset)
    :noindex:

    :func:`~odtk.preprocessing.change.change_to_binary`, :func:`~odtk.preprocessing.change.change_to_label`
    and :func:`~odtk.preprocessing.change.change_to_one_hot` changes the encoding method for occupancy
    in the ``odtk.data.dataset.Dataset`` object.:

        >>> import odtk
        >>> import numpy as np
        >>> data = np.array([[1, 2], [3, 4]])
        >>> occupancy = np.array([[0], [3]])
        >>> dataset = odtk.data.Dataset()
        >>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["name 1", "name 2"])
        >>> dataset.occupancy
        array([[0.],
               [3.]])
        >>> odtk.preprocessing.change_to_one_hot(dataset)
        >>> dataset.occupancy
        array([[1., 0., 0., 0.],
               [0., 0., 0., 1.]])
        >>> odtk.preprocessing.change_to_label(dataset)
        >>> dataset.occupancy
        array([[0],
               [3]], dtype=int64)
        >>> odtk.preprocessing.change_to_binary(dataset)
        >>> dataset.occupancy
        array([[0],
               [1]], dtype=int64)

odtk.preprocessing.fill module
------------------------------

.. py:function:: odtk.preprocessing.fill.fill(dataset)
    :noindex:

    :func:`~odtk.preprocessing.fill.fill` function helps impute all missing data
    in the ``odtk.data.dataset.Dataset`` object.:

        >>> import odtk
        >>> import numpy as np
        >>> data = np.array([[1, np.nan], [100, 4], [200, np.nan], [20, np.nan]])
        >>> occupancy = np.array([[0], [3], [4], [7]])
        >>> dataset = odtk.data.Dataset()
        >>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["name 1", "name 2"])
        >>> dataset.data
        array([[  1.,  nan],
               [100.,   4.],
               [200.,  nan],
               [ 20.,  nan]])
        >>> odtk.preprocessing.fill(dataset)
        >>> dataset.data
        array([[  1.,   4.],
               [100.,   4.],
               [200.,   4.],
               [ 20.,   4.]])

odtk.preprocessing.ontology module
----------------------------------

.. py:function:: odtk.preprocessing.ontology.ontology(dataset)
    :noindex:

    :func:`~odtk.preprocessing.ontology.ontology` function can be used to change the feature names
    in the ``odtk.data.dataset.Dataset`` object.:

        >>> import odtk
        >>> import numpy as np
        >>> data = np.array([[1, 1], [2, 4]])
        >>> occupancy = np.array([[0], [3]])
        >>> dataset = odtk.data.Dataset()
        >>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["Tem.", "damp"])
        >>> odtk.preprocessing.ontology(dataset)
        ['temperature', 'damper']
        >>> dataset.feature_list
        ['temperature', 'damper']

odtk.preprocessing.outlier module
---------------------------------

.. py:function:: odtk.preprocessing.outlier.remove_outlier(dataset, auto_fill=True, ratio=1.5)
    :noindex:

    :func:`~odtk.preprocessing.outlier.remove_outlier` function removes outliers that exist in
    the ``odtk.data.dataset.Dataset`` object and fills in the gap that will be produced.:

        >>> import odtk
        >>> import numpy as np
        >>> data = np.array([[1, 1], [2, 4], [200, 4], [3, 7]])
        >>> occupancy = np.array([[0], [3], [4], [7]])
        >>> dataset = odtk.data.Dataset()
        >>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["name 1", "name 2"])
        >>> dataset.data
        array([[  1.,   1.],
               [  2.,   4.],
               [200.,   4.],
               [  3.,   7.]])
        >>> odtk.preprocessing.remove_outlier(dataset)
        >>> dataset.data
        array([[1., 1.],
               [2., 4.],
               [2., 4.],
               [3., 7.]])

odtk.preprocessing.upsample module
----------------------------------

.. py:function:: odtk.preprocessing.upsample.upsample(dataset, target_frequency, algorithm='linear')
    :noindex:
.. py:function:: odtk.preprocessing.downsample.downsample(dataset, target_frequency, algorithm='mean')
    :noindex:

    :func:`~odtk.preprocessing.upsample.upsample` and :func:`~odtk.preprocessing.downsample.downsample` increase and
    decrease the number of data points (i.e., the number of rows) in the ``odtk.data.dataset.Dataset`` object.:

        >>> import odtk
        >>> import numpy as np
        >>> data = np.array([[1, 2], [100, 4], [200, 6]])
        >>> occupancy = np.array([[0], [3], [4]])
        >>> dataset = odtk.data.Dataset()
        >>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["name 1", "name 2"])
        >>> dataset.data
        array([[  1.,   2.],
               [100.,   4.],
               [200.,   6.]])
        >>> dataset.time_column_index = 0
        >>> odtk.preprocessing.upsample(dataset, 10)
        >>> dataset.data
        array([[  1.        ,   2.        ],
               [ 11.        ,   2.22222222],
               [ 21.        ,   2.44444444],
               [ 31.        ,   2.66666667],
               [ 41.        ,   2.88888889],
               [ 51.        ,   3.11111111],
               [ 61.        ,   3.33333333],
               [ 71.        ,   3.55555556],
               [ 81.        ,   3.77777778],
               [ 91.        ,   4.        ],
               [101.        ,   4.2       ],
               [111.        ,   4.4       ],
               [121.        ,   4.6       ],
               [131.        ,   4.8       ],
               [141.        ,   5.        ],
               [151.        ,   5.2       ],
               [161.        ,   5.4       ],
               [171.        ,   5.6       ],
               [181.        ,   5.8       ],
               [191.        ,   6.        ]])
        >>> odtk.preprocessing.downsample(dataset, 60)
        >>> dataset.data
        array([[  1.        ,   2.        ],
               [ 61.        ,   3.33333333],
               [121.        ,   4.6       ],
               [181.        ,   5.8       ]])
