odtk.stats packageÂ¶

This package contains functions for analyzing odtk.data.dataset.Dataset and assessing its quality. odtk.stats.analysis.analysis() runs all sorts of analysis automatically.

odtk.stats.dropout_rate.dropout_rate(dataset, dataset_level=False)

dropout_rate() function computes the percentage of data points missing in the odtk.data.dataset.Dataset object.

odtk.stats.frequency.frequency(dataset, dataset_level=True)

frequency() function computes the average sampling frequency of all features in the odtk.data.dataset.Dataset object.

odtk.stats.gap_detect.gap_detect(dataset, threshold, sensor_level=False)

gap_detect() function returns all gaps in the odtk.data.dataset.Dataset object.

We say that there is a Gap when the difference between timestamps of two successive values exceeds some threshold.

odtk.stats.occupancy_evaluation.occupancy_distribution_evaluation(dataset, dataset_level=True)

occupancy_distribution_evaluation() function computes the distribution of different occupancy states in the odtk.data.dataset.Dataset object.

odtk.stats.uptime.uptime(dataset, threshold, gaps=None)

uptime() function computes the length of time a sensor reported values. It differs from the dropout_rate() in that it captures the length of the time interval in which data points were available in the odtk.data.dataset.Dataset object.

A sample code snippet for using the package is here:

>>> import odtk
>>> dataset = odtk.data.load_sample("umons-all")
>>> from pprint import pprint
>>> # ===== Test dropout rate =====
>>> pprint(odtk.stats.dropout_rate(dataset, dataset_level=False))
{'datatest': 0.0, 'datatest2': 0.0, 'datatraining': 0.0}
>>> pprint(odtk.stats.dropout_rate(dataset, dataset_level=True))
0.0
>>> # ===== Test frequency =====
>>> pprint(odtk.stats.frequency(dataset, dataset_level=False))
{'datatest': 60.0, 'datatest2': 60.0, 'datatraining': 60.0}
>>> pprint(odtk.stats.frequency(dataset, dataset_level=True))
60.0
>>> # ===== Find all gaps =====
>>> pprint(odtk.stats.gap_detect(dataset, 65, sensor_level=False))
{'datatest': [], 'datatest2': [], 'datatraining': []}
>>> pprint(odtk.stats.gap_detect(dataset, 65, sensor_level=True))
