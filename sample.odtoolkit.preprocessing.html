odtoolkit.preprocessing package¶

This package contains functions that are useful for modifying values in odtoolkit.data.dataset.Dataset. odtoolkit.preprocessing.auto_clean.auto_clean() automatically runs all preprocessing functions.

odtoolkit.preprocessing.change module¶

odtoolkit.preprocessing.change.change_to_binary(dataset)
odtoolkit.preprocessing.change.change_to_label(dataset)
odtoolkit.preprocessing.change.change_to_one_hot(dataset)

change_to_binary(), change_to_label() and change_to_one_hot() changes the encoding method for occupancy in the odtoolkit.data.dataset.Dataset object.:

>>> import odtoolkit
>>> import numpy as np
>>> data = np.array([[1, 2], [3, 4]])
>>> occupancy = np.array([[0], [3]])
>>> dataset = odtoolkit.data.Dataset()
>>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["name 1", "name 2"])
>>> dataset.occupancy
array([[0.],
       [3.]])
>>> odtoolkit.preprocessing.change_to_one_hot(dataset)
>>> dataset.occupancy
array([[1., 0., 0., 0.],
       [0., 0., 0., 1.]])
>>> odtoolkit.preprocessing.change_to_label(dataset)
>>> dataset.occupancy
array([[0],
       [3]], dtype=int64)
>>> odtoolkit.preprocessing.change_to_binary(dataset)
>>> dataset.occupancy
array([[0],
       [1]], dtype=int64)

odtoolkit.preprocessing.fill module¶

odtoolkit.preprocessing.fill.fill(dataset)

fill() function helps impute all missing data in the odtoolkit.data.dataset.Dataset object.:

>>> import odtoolkit
>>> import numpy as np
>>> data = np.array([[1, np.nan], [100, 4], [200, np.nan], [20, np.nan]])
>>> occupancy = np.array([[0], [3], [4], [7]])
>>> dataset = odtoolkit.data.Dataset()
>>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["name 1", "name 2"])
>>> dataset.data
array([[  1.,  nan],
       [100.,   4.],
       [200.,  nan],
       [ 20.,  nan]])
>>> odtoolkit.preprocessing.fill(dataset)
>>> dataset.data
array([[  1.,   4.],
       [100.,   4.],
       [200.,   4.],
       [ 20.,   4.]])

odtoolkit.preprocessing.ontology module¶

odtoolkit.preprocessing.ontology.ontology(dataset)

ontology() function can be used to change the feature names in the odtoolkit.data.dataset.Dataset object.:

>>> import odtoolkit
>>> import numpy as np
>>> data = np.array([[1, 1], [2, 4]])
>>> occupancy = np.array([[0], [3]])
>>> dataset = odtoolkit.data.Dataset()
>>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["Tem.", "damp"])
>>> odtoolkit.preprocessing.ontology(dataset)
['temperature', 'damper']
>>> dataset.feature_list
['temperature', 'damper']

odtoolkit.preprocessing.outlier module¶

odtoolkit.preprocessing.outlier.remove_outlier(dataset, auto_fill=True, ratio=1.5)

remove_outlier() function removes outliers that exist in the odtoolkit.data.dataset.Dataset object and fills in the gap that will be produced.:

>>> import odtoolkit
>>> import numpy as np
>>> data = np.array([[1, 1], [2, 4], [200, 4], [3, 7]])
>>> occupancy = np.array([[0], [3], [4], [7]])
>>> dataset = odtoolkit.data.Dataset()
>>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["name 1", "name 2"])
>>> dataset.data
array([[  1.,   1.],
       [  2.,   4.],
       [200.,   4.],
       [  3.,   7.]])
>>> odtoolkit.preprocessing.remove_outlier(dataset)
>>> dataset.data
array([[1., 1.],
       [2., 4.],
       [2., 4.],
       [3., 7.]])

odtoolkit.preprocessing.upsample module¶

odtoolkit.preprocessing.upsample.upsample(dataset, target_frequency, algorithm='linear')
odtoolkit.preprocessing.downsample.downsample(dataset, target_frequency, algorithm='mean')

upsample() and downsample() increase and decrease the number of data points (i.e., the number of rows) in the odtoolkit.data.dataset.Dataset object.:

>>> import odtoolkit
>>> import numpy as np
>>> data = np.array([[1, 2], [100, 4], [200, 6]])
>>> occupancy = np.array([[0], [3], [4]])
>>> dataset = odtoolkit.data.Dataset()
>>> dataset.add_room(data, occupancy=occupancy, room_name="test", header=["name 1", "name 2"])
>>> dataset.data
array([[  1.,   2.],
       [100.,   4.],
       [200.,   6.]])
>>> dataset.time_column_index = 0
>>> odtoolkit.preprocessing.upsample(dataset, 10)
>>> dataset.data
array([[  1.        ,   2.        ],
       [ 11.        ,   2.22222222],
       [ 21.        ,   2.44444444],
       [ 31.        ,   2.66666667],
       [ 41.        ,   2.88888889],
       [ 51.        ,   3.11111111],
       [ 61.        ,   3.33333333],
       [ 71.        ,   3.55555556],
       [ 81.        ,   3.77777778],
       [ 91.        ,   4.        ],
       [101.        ,   4.2       ],
       [111.        ,   4.4       ],
       [121.        ,   4.6       ],
       [131.        ,   4.8       ],
       [141.        ,   5.        ],
       [151.        ,   5.2       ],
       [161.        ,   5.4       ],
       [171.        ,   5.6       ],
       [181.        ,   5.8       ],
       [191.        ,   6.        ]])
>>> odtoolkit.preprocessing.downsample(dataset, 60)
>>> dataset.data
array([[  1.        ,   2.        ],
       [ 61.        ,   3.33333333],
       [121.        ,   4.6       ],
       [181.        ,   5.8       ]])
