Welcome to the ODToolkit API Documentation Page¶

ODToolkit is a Python package that offers open-source implementation of various occupancy detection models and allows for comprehensive evaluation of these models on publicly available data sets collected from residential and commercial buildings. ODToolkit’s data processing and evaluation pipeline is shown below.

Pre-request: Python 3.4+.
